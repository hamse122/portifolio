## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```



## Customization

### Colors

Edit `tailwind.config.ts` to customize the color palette:
- `accent-from`: #0AB6BC
- `accent-to`: #045C43
- `accent-neon`: #00FFC6
- `background-dark`: #000000
- `background-navy`: #021B23

### Content

Update project information in:
- `components/Projects.tsx` - Add your projects
- `components/Hero.tsx` - Modify hero text
- `components/About.tsx` - Update about section
- `components/Skills.tsx` - Add/remove skills

## Deployment

The easiest way to deploy is using [Vercel](https://vercel.com):

```bash
npm i -g vercel
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
